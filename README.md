Projeto fullstack realizado em mais modulo do curso de frontend,
novamente usamos nodeJs no backend ( express, cors, mongoDb, axios ) e no frontend ReactJs
( react-router-dom , react-toastify )

É um projeto para adicionar memorias, onde você adiciona uma foto, comentário, descrição, etc... 
Também foi adicionado um botão para apagar a memoria que foi adicionada ( proximo passo é excluir o comentário ).
